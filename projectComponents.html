Project Components

  • Every user has to login into the homepage atleast once before using the website features. If the user is new to the website then he/she can set up a new account by signing up and providing necessay details like email-ID, username, educational qualification etc and setting up a password. Username and user email-ID should be unique for avoiding unwanted conflicts.
  • After logging into the website successfully the homepage is displayed. Its UI enables users to host, register, attempt and review the quiz by a single button click. All the quizes hosted on the website are all displayed using carousel on the homepage which contains all the necessay information about the quiz content and conduct. There is an additional feature in foot section of the homepage where the number of current users in the website and number of quizes hosted are displayed dynamically. The users can also veiw their profile and the website overview. Logout feature is also added where the user can logout from the website if required and in the later stage can reuse by logging in.
  • Host a quiz: This page takes the questions from the host and stores it in the userbase. The host is asked to enter the details like Quiz name, date of the exam, start time, end time, eligibility etc Then the host is asked to add quetions. We are allowing multiple choice questions with 2-5 options for each question. The host has to give the question and after giving number of options, the blanks for entering the choices appear dynamically. The host must put enter the correct choice against which we will evalute the quiz. We are planning to implement taking images of questions in the coming days.
  • The quiz portal has many subtle features which enables the users to become scrupulous about their responses and attempt to thier level best.
